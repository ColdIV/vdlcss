Getting started

To add vdlcss to your site, download it and include it in your <head>.
Don't forget to add the meta tag to allow responsive styling.

Download 
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<link rel="stylesheet" type="text/css" href="vdlcss-min.css">

Features

Classless Base

While there are many helpful classes, the base is classless!

No need to add a "button" class to a <button>, it's clear what it is!

Grid

Similar to Bootstrap, there is a 12-cell grid system.

However, the grid system is based on css grids and the 12-cells are always relative to the current container.

Rows and Columns

row and column are flex-based and let you position your divs with ease!

The class names describe what they do within the context. Which is why a single div within a row bottom right looks the same as a single div within a column bottom right.

Breakpoints

To use classes for a specific screen size, you can chose between 6 different breakpoints, ranging from xs to xxl.

Breakpoint specific classes can be applied by using the breakpoint as a prefix, for example: md:pr-1 will add a padding on the right side for the breakpoints md and above.
